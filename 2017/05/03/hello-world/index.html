HEXO + Github 自己搭建一个博客(win端)

环境准备

1、安装Node

2、安装Git

3、推荐两款后面会用到的文本编辑器：Sublime & Cmd Markdown（支持同步编辑）

安装 HEXO

要在Node和Git都安装好的前提下

1、打开windows控制台，输入如下命令：

npm install -g hexo

2、初始化

  • 创建一个存放你这个博客的文件夹，如：Blog
  • 控制台cd到Blog里执行如下命令：
    hexo init
    至此，hexo安装完毕

继续在Blog目录下执行如下命令，生成HelloWorld静态页面：

hexo generate

启动本地服务，进行文章预览调试，命令如下：

hexo server

然后浏览器输入 http://localhost:4000 就能看到

关联 Github

1、注册账号

2、建立与用户名对应的仓库，仓库名必须为 your_user_name.github.io

接下来有两种方式可以将你的编辑内容上传至github：

  • 配置SSH keys
  • 建立Repository
添加 SSH Keys

打开 Git bash，依次输入如下命令：

  • 首先设置用户名和邮箱：

    git config –global user.email”your_email”
    git config –global user.name”your_user_name”

  • 生成密钥
    ssh-keygen -t rsa -C “your_email”

在你执行命令的当前目录下会得到两个文件 id_rsa 和 id_rsa.pub
然后打开 .pub 文件，复制其中内容到 Add SSH Key
再将id_rsa和id_rsa.pub两个文件拷贝到.ssh目录

最后可以验证一下：

ssh -T git@github.com
建立Repository

首先打开你的Blog目录下的_config.yml文件，找到depoly:项，改成如下：

deploy: 
  type: git
  repository: https://github.com/lusonhey/lusonhey.github.io.git
  branch: master

千万注意：: 后面要有空格

最后输入如下命令完成git部署：

npm install hexo-deployer-git --save

以后写完博客两个命令就可以同步到github上：

hexo generate
hexo deploy
